# 仅记录我毕业后的一些工作经历


从毕业就一直在广州这边，都没想过去外地发展（或许是没有机会）

## 广州稷生科技有限公司

2022.04 - 2022.12（模糊时间）

当时还没毕业，自己出来找实习时没有任何信心，一是认为自己不是什么名牌大学，二是在校园招聘会上被八股文虐了一遍，所以后来出来社会上找实习时也是非常曲折。

起初第一次步入社会去找工作，很多都不懂，甚至租房子都是在家人的帮助下租下来的，在大三时自学了vue，找工作时清一色的vue技术栈，当初约面试并不难，但是不知道出于什么原因，最后到薪资那一块都不高（不管面试过程是否顺利的都一样），先听我对于不高的定义是什么：
 1. 能解决我的房租以及水电
 2. 除去房租和水电后能解决我的温饱（我只算了一天两顿的猪脚饭，那时租房楼下是12块）

 我前前后后面了10个公司左右，那时候面试一趟都觉得累（内心脆弱吧那时候），其中包括一些小程序开发，后台的开发。最后面试了《广州稷生科技有限公司》，那时给的工资是虽然不高，当时因为觉得没工作，认为先找一份过过日子先，就接下了offer。

 该公司当时是做一个养老院（应该是，太久了）的项目，是政府的项目，老板不是技术出身，我的工作内容是后台管理系统的开发，其实架构哪些都已经搭好了，我和一些同为实习生的伙伴只负责子业务的页面开发，这也是我第一次在工作中使用vue，由于当时公司没有专门的UI和测试，所以很多实现细节都由我们自己决定，最终由老板决定是否更改，虽然听起来很荒唐，但后面我了解到是因为需要赶工的无奈之举，实习生没什么好的，就是爱思考，所以我尽可能地去想老板喜欢什么样地交互以及动效，其实那段时间我也学习了很多我没接触过地知识。

 最让我措不及防地，我在元旦前提了离职，理由也简单，因为我住的比较远。只能提前回家过年，在过年期间，我突然收到《广州稷生科技有限公司》的电话问我能否可以回去继续工作，我思考之后还是委婉拒绝了。

 总的来说还是一段比较愉快的实习，期间学校导师还来巡查是属实没想到（社死现场）
